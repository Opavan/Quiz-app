# Quiz App

## Description
An interactive and responsive Quiz App built using **HTML, CSS, and JavaScript**.  
Users can attempt multiple-choice questions, receive instant feedback on their answers, and track their scores in real-time. The clean and intuitive interface ensures a smooth experience on both desktop and mobile devices.

---

## Features
- Multiple-choice questions
- Real-time feedback for correct/incorrect answers
- Score tracking at the end of the quiz
- Responsive design for desktop and mobile
- Interactive and user-friendly UI

---

## Tech Stack
- **HTML** – Structure of the app
- **CSS** – Styling and responsive design
- **JavaScript** – Functionality and interactivity
